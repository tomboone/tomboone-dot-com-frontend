# Tom Boone Portfolio Frontend

React frontend for [tomboone.com](https://tomboone.com/) portfolio website.

## Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Environment setup:**
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

3. **Run development server:**
   ```bash
   npm run dev
   ```
